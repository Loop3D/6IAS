Spatial Random Forest - Clustering

  • Geoscience data typically hold informative statistical and spatial patterns. Consequently, such statistical and spatial patterns should be distinguishable from a randomly generated version of themselves.
  • Original data is labelled as 1 and synthetic data is label as 0.

Advantages and Disadvantage

  • Interpretability
  • Handling categorical data
  • Handling NA values
  • Significant computation resources
  • Alternative methods available
    • Filtering + Neural Network
    • Generative and Adversarial Network

Scale and Resolution

  • Scale doesn’t significantly impact the results as long as all training data are on the same scale.

  • Higher resolution requires more computing power, including both memory and processing cores.

  • The number of layers directly influences the required memory and processing cores, with more layers requiring more resources.

Computation Issue

Methods

  • Implement sampling from the original data.
  • Train the model using this sampled data.

Potential drawbacks

  • Information loss
  • Potential bias introduced by sampling
